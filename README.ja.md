# ar-handtrack-torus

このリポジトリでは、Three.jsを使ったARハンドトラッキングを実装しています。手の動きを検知し、トーラスオブジェクトを操作できます。

## デモ
https://code4fukui.github.io/ar-handtrack-torus/

## 機能
- Webカメラ映像上にARでトーラスオブジェクトを表示
- 手の動きでトーラスオブジェクトを掴んだり移動できる
- その他のアセットも追加で表示可能 (ホットサンド、ケーキ、海鮮丼など)

## 使い方
ブラウザでデモページにアクセスし、Webカメラの使用を許可すると、ARデモが始まります。手でトーラスオブジェクトを掴んで操作できます。
